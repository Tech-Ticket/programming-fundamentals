🤖 What is Programming?

Simply put, programming is the way we talk to computers.

Computers are incredibly fast and powerful, but they are also completely blind and not very smart on their own. They can't do anything unless a human tells them exactly what to do.

When you learn to program, you are learning to write a set of instructions that a computer can understand. You already use programmed things every single day:

  • 🎮 Video Games: Code tells the game what happens when you press the 'Jump' button.
  • 📱 Mobile Apps: Code tells TikTok or Instagram how to show you the next video.
  • 🌐 Websites: Code built the web browser you are looking at right now.
  • 🤖 Robots: Code tells Mars Rovers how to drive on another planet!

🧠 How Computers Follow Instructions

No matter how complicated a program is (even a 3D video game!), it usually follows a very simple three-step cycle:

  1. Input: You give the computer some information (like typing on a keyboard or clicking a mouse).
  2. Process: The computer thinks about it and does some math or logic.
  3. Output: The computer gives you a result (like displaying a picture on the screen or playing a sound).

📝 What is an Algorithm?

An algorithm is just a fancy word for a step-by-step list of instructions to solve a problem or get something done.

You use algorithms all the time in real life!

🥪 Algorithm for making a sandwich

  1. Take out two slices of bread.
  2. Spread peanut butter on one slice.
  3. Spread jelly on the other slice.
  4. Smash them together.
  5. Eat!

📸 Algorithm for Instagram

  1. Open the Instagram app.
  2. Type your username.
  3. Type your secret password.
  4. Tap the "Login" button.
  5. Scroll through memes.

In programming, we write algorithms using a coding language so the computer knows what to do.

🚀 Introduction to C++ & Your Coding Setup

There are hundreds of programming languages, but in this course, we are learning C++ (pronounced "Cee-Plus-Plus").

Why C++? C++ is incredibly fast and powerful. Because of its speed, it is used to build things where performance is critical. For example, the Unreal Engine (which powers games like Fortnite), web browsers like Google Chrome, and even the software running on space shuttles are built with C++!

🛠️ The Code Editor: Visual Studio Code

To write our code, we use a special program called a Code Editor. Think of it like Microsoft Word, but designed specifically for typing code instead of essays.

We will be using Visual Studio Code (VS Code). It highlights our code with pretty colors to make it easier to read and helps us catch mistakes.

💻 Your First C++ Program

It is a long-standing tradition in the programming world that your very first program should simply make the computer say "Hello" to the world. Let's do it!

main.cpp 
#include <iostream>
using namespace std;

int main()
{
    cout << "Hello TechTicket!";
    return 0;
}

🔍 Step-by-Step Code Breakdown

Looks like an alien language? Don't worry! Let's translate it line by line.

  • #include <iostream>
    This is like grabbing your toolbox before you start working. `iostream` stands for Input/Output Stream. We need this tool to make text appear on the screen.
  • using namespace std;
    This is a shortcut. Without it, we would have to type `std::` before a lot of words. It saves us from extra typing!
  • int main()
    This is the starting line of the race! Every single C++ program must have a `main` function. When the computer runs your code, it immediately looks for `main()` and starts reading from the opening curly brace { to the closing one }.
  • cout << "Hello TechTicket!";
    cout stands for "Character OUTput". It tells the computer to print text to the screen. The two arrows << are pushing the words "Hello TechTicket!" out to the screen. Notice the semicolon ; at the end? That’s the C++ version of a period at the end of a sentence. Never forget your semicolons!
  • return 0;
    This tells the computer: "We reached the end of the program, and everything went perfectly (zero errors)!"

👨‍🍳 Real-Life Analogy: The Literal Chef

Programming is like writing a recipe for a very obedient, but extremely literal chef. If you tell the chef "put peanut butter on bread", but you forget to say "open the jar first", the chef will try to smash a glass jar onto the bread! Computers are the same. They only do exactly what you type. If you forget a semicolon ;, the computer gets confused and stops cooking.

🏋️‍♂️ Practice Exercises

Now it's your turn! Open Visual Studio Code and try modifying the code to complete these exercises.

Exercise 1: Say your name!

Change the code so that instead of "Hello TechTicket!", it prints "Hello my name is [Your Name]".

Look at line 6. Change the text inside the quotation marks `""`. Don't delete the quotation marks!

Exercise 2: How old are you?

Delete your previous code line and write a new one that prints your age. For example: "I am 14 years old".

Your code inside `main()` should look like this:
cout << "I am 14 years old";

Exercise 3: Gamer Check

Write a program that outputs your favorite video game to the screen.

Use `cout` and `<<` just like before:
cout << "My favorite game is Minecraft!";
Make sure you put the semicolon `;` at the end!

🔥 Challenge Exercise

Ready for something a little harder?

Write a program that prints a short introduction about yourself using multiple cout statements.

Pro Tip: If you want your text to move to a new line (like pressing "Enter" on your keyboard), you can add << endl; at the end of your cout statement. `endl` stands for "End Line".

cout << "Hello! My name is Alex." << endl;
cout << "I am learning C++." << endl;
