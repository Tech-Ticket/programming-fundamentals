Session 8

Arrays and Working with Multiple Values

🎯 Session Overview

Welcome to Session 8! You've learned how to create variables to store single pieces of data. But what happens when you need to store the scores of an entire classroom of students?

In this session, we will learn how to group data together. You will learn:

  • What an array is and why it's better than using a hundred normal variables.
  • How to create an array in C++ to store lists of data.
  • How to grab specific items out of that list using an index (and why programmers always start counting at zero!).
  • How to combine arrays and loops to build powerful data-processing programs.

⏪ Session 7 Recap

In our last session, we talked about organizing our code using Functions.

  • Functions are custom mini-machines we build to do specific tasks.
  • They stop us from writing the same block of code over and over again.
  • We call a function from inside main() to make it run.

Functions organize our code. Today, we learn how to organize our data.

😩 The Problem: Too Many Variables!

Imagine you are making a game, and you need to store the top 5 high scores. Right now, you would have to do this:

int score1 = 10;
int score2 = 20;
int score3 = 30;
int score4 = 40;
int score5 = 50;

That is messy! Now imagine if the game had 10,000 players. You cannot write 10,000 different variables. The code would be impossible to read and terrible to type. We need a way to group all these numbers under one single name.

📦 What is an Array?

An array is a collection of variables of the same type, stored together in a single row under one name.

Instead of having 5 separate cardboard boxes scattered around your room, an array is like a single organizer box with 5 neat compartments inside it.

⚠️ The Golden Rule of Programming

Look closely at the diagram above. Notice how the compartments are numbered: 0, 1, 2, 3, 4. Computers always start counting at ZERO! The first item in an array is at Index 0. The second item is at Index 1. This is confusing at first, but every programmer in the world has to get used to it!

💻 Creating and Accessing Arrays in C++

Let's turn that diagram into real C++ code. We will create the array, and then print the first and second numbers from it.

array_basics.cpp 
#include <iostream>
using namespace std;

int main()
{
    // Create an integer array named 'numbers' that holds 5 items.
    // We use curly braces {} to put the data inside it immediately.
    int numbers[5] = {10, 20, 30, 40, 50};

    // Access the elements using square brackets []
    cout << "The first number is: " << numbers[0] << endl;
    cout << "The second number is: " << numbers[1] << endl;

    return 0;
}

Code Breakdown:

  • int numbers[5] tells the computer: "Make an array named numbers that holds exactly 5 integers."
  • The numbers inside the square brackets [ ] are called the Index.
  • numbers[0] opens the first compartment and finds the 10.

🏫 Real-Life Analogy: Hallway Lockers

Think about the row of lockers in a school hallway. They are all identical boxes lined up against the wall.

  • The whole row of lockers is the Array.
  • The number painted on the outside of each locker is the Index (Locker 0, Locker 1, Locker 2).
  • The books and backpacks stored inside are the Values.

If you tell a friend, "My math book is in Locker 3", they know exactly where to go. In C++, you say cout << lockers[3]; to get the same result!

🔄 The Perfect Match: Loops and Arrays

Printing numbers[0], numbers[1], and so on manually is annoying. But remember the for loop we learned? A loop is basically an automatic counter.

If we use our loop counter variable (i) as the array Index, the loop will automatically walk through the entire array for us!

array_loop.cpp 
#include <iostream>
using namespace std;

int main()
{
    int numbers[5] = {10, 20, 30, 40, 50};

    // Start i at 0 (the first index).
    // Keep going while i is strictly less than 5 (which means it stops at 4!).
    for(int i = 0; i < 5; i++)
    {
        cout << numbers[i] << endl;
    }

    return 0;
}

How the loop sees the array:

Notice that our loop condition is i < 5. Since we start at 0, the numbers 0, 1, 2, 3, 4 are exactly 5 loops! The loop stops before i becomes 5, which is perfect because numbers[5] does not exist!

🏠 Homework Assignment

Write a program that acts like a teacher's gradebook!

  1. Create a float array called scores that holds 5 test scores (like 95.5, 80.0, etc.).
  2. Write a for loop to print out every score.
  3. Make the output look nice! It should print: "Student 1 score: 95.5", then "Student 2 score: 80.0".

Tricky Hint: The array index is i (which starts at 0), but your printout says "Student 1". How can you print the number 1 when the loop is on 0? You might need to print (i + 1)!

📝 Session Summary

  • An Array is a single variable that can hold a list of multiple values of the same type.
  • Arrays are created by putting the size inside square brackets: int scores[5];
  • Each compartment in an array is numbered, starting at 0. This number is called the Index.
  • You access the data by putting the index in brackets: scores[0] gives you the first item.
  • Loops and Arrays are best friends. Using the loop's i variable as the index allows you to quickly process every item in an array automatically!
