# TechTicket – Programming Fundamentals with C++

An interactive beginner-friendly programming course designed for students aged **11–17** to learn the fundamentals of programming using **C++**.

This course introduces core programming concepts through **visual explanations, interactive examples, coding exercises, and mini projects**.
Each session is presented as an **interactive web page** so students can review the material anytime.

## 📚 Course Overview

The **Programming Fundamentals with C++** course helps students understand how programming works and how to build real programs step by step.

Students will learn:

* How computers follow instructions
* Variables and data types
* User input and output
* Operators and calculations
* Decision making with `if` statements
* Loops and repetition
* Functions and organizing code
* Arrays and working with multiple values
* Problem solving with programming
* Building a final mini project

---

## 🧠 Course Sessions

1. **Session 1 — Introduction to Programming**
2. **Session 2 — Variables**

6. **Session 6 — Loops**
7. **Session 7 — Functions**
8. **Session 8 — Arrays**

* Homework tasks
