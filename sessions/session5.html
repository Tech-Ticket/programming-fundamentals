Session 5

Decision Making with if and else

🎯 Session Overview

Welcome back! So far, our programs have been a little simple—they run straight from the top line to the bottom line without ever stopping to think. But what if we want our program to be smart and make choices?

In this session, you will learn:

  • How programs check conditions and make decisions.
  • How to use the if statement to run code only when a rule is met.
  • How to use else to give the program a backup plan.
  • How to build fun, interactive programs like a password checker or a grade calculator!

🔄 Quick Recap: Session 4

In our last session, we learned about comparison operators like >, <, and ==.

Remember that when you compare two things, the computer gives you an answer of either true or false.

10 > 5   // This is TRUE!
10 == 20  // This is FALSE!

Today, we will use these "true or false" answers to tell our program what to do next.

🤔 Why Do Programs Need to Make Decisions?

Think about the apps and games you use every day. They constantly make decisions based on what you do!

  • 🎮 In a Video Game: If your health reaches 0, it shows the "Game Over" screen.
  • 🏫 In School software: If your score is above 50, it says "Pass", otherwise it says "Fail".
  • 📱 Logging into Instagram: If your password is correct, it lets you in. If it is wrong, it stops you.

Without the ability to make decisions, programming would be useless. Programs must check conditions before acting.

🚦 Introduction to if Statements

The if statement is how we tell C++ to check a rule. It translates perfectly to English: "If this condition is true, then do this action."

Here is what the syntax (the grammar of the code) looks like:

if (condition)
{
    // Action goes here!
    // This code ONLY runs if the condition is TRUE.
}

A Simple Example

Let's write a program that asks for your age. If you are 18 or older, it will welcome you as an adult.

check_age.cpp 
#include <iostream>
using namespace std;

int main()
{
    int age;
    cout << "Enter your age: ";
    cin >> age;

    // The Decision Making Part!
    if (age >= 18)
    {
        cout << "You are an adult!" << endl;
    }

    return 0;
}

Step-by-Step Breakdown:

  1. We create an int variable called age.
  2. We use cin to get a number from the user.
  3. The if (age >= 18) checks: Is the number they typed greater than or equal to 18?
  4. If they typed 20, the condition is true, and the program prints "You are an adult!".
  5. If they typed 14, the condition is false, the program skips the curly braces {}, and prints nothing!

🛤️ Using if and else

In our previous example, if the user was 14, the program just ended silently. That's not very helpful!

Sometimes, we need a "Plan B". That is exactly what else does. It tells the computer: "If the first condition is false, do this backup action instead."

school_grades.cpp 
#include <iostream>
using namespace std;

int main()
{
    int score;
    cout << "Enter your score out of 100: ";
    cin >> score;

    if (score >= 50)
    {
        // This runs if score is 50 or higher
        cout << "Congratulations! You passed!" << endl;
    }
    else
    {
        // This runs if score is LESS than 50
        cout << "Oh no! You failed!" << endl;
    }

    return 0;
}

Here is what the computer's "brain" looks like when it runs an if / else statement. It acts like a fork in the road.

🕹️ Real-Life Analogy: The Arcade

Imagine you are at a video game arcade. You walk up to a racing game, and it says "Insert 3 Coins to Play".

Here is exactly how the arcade machine's "brain" thinks:

  • IF you insert 3 or more coins:
    • ✔️ The machine turns on the steering wheel and lets you play.
  • ELSE (if you inserted 2 or less):
    • ❌ The screen flashes "Not enough coins!" and does not start.

Your C++ programs use if and else to act exactly like that smart machine!

Ask the user to enter a number. If the number is greater than 10, print "Number is greater than 10". (You don't need an else for this one!)

Exercise 2: Movie Theater Ticket

Ask the user for their age. If the age is 18 or more, print "Adult Ticket". Else, print "Minor Ticket".

Use the >= operator for "greater than or equal to 18". Remember to use the curly braces {} for both the if block and the else block!

Exercise 3: The Grader

Ask the user for their math score. If their score is 50 or higher, print "Pass". Else, print "Fail".

if (score >= 50) {
    cout << "Pass";
} else {
    cout << "Fail";
}

🔥 Challenge Exercise

This one is tricky! Write a program that asks the user for a number, and then tells them if the number is Even or Odd.

Hint: Use the modulus operator (%). If you divide an EVEN number by 2, what is the remainder? It's always 0!

Use this condition inside your if statement:

if (number % 2 == 0)

Remember to use double equals == when checking if things are the same!

📝 Session Summary

  • Conditional Statements give your program the power to make choices.
  • The if statement checks a rule. If the rule is true, it runs the code inside its curly braces {}.
  • The else statement is the backup plan. It runs only if the if condition was false.
  • You use comparison operators (like ==, >, <=) inside the parentheses () to set the rule.
  • Decision making is the foundation of almost all smart software, from video games to artificial intelligence!
