What You Will Learn

Designed specifically for students aged 11–17, this course breaks down complex coding concepts into fun, bite-sized lessons.

Programming Basics

Understand how computers read and execute instructions step by step.

Variables & Data

Learn how programs store information like scores, names, and passwords.

Logic & Decisions

Teach your programs to make smart choices using if and else conditions.

Loops & Repetition

Save time by making the computer do the boring, repetitive work for you.

Functions & Arrays

Organize your code like a pro and manage large lists of data efficiently.

Real Projects

Apply your skills by building calculators, quizzes, and mini-games.

Your Learning Journey

A structured step-by-step path from beginner to builder.

Step 1: The Basics

Sessions 1-4. Learn the syntax, how to store data, and basic math.

Step 2: Logic Flow

Sessions 5-6. Control your program using decisions and loops.

Step 3: Structure

Sessions 7-8. Organize your code with functions and arrays.

Step 4: Mastery

Sessions 9-10. Problem solving and building your final mini-project.
