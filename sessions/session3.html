🧩 Why Do Data Types Exist?

Imagine organizing your closet. You wouldn't put dirty shoes in the same drawer as your clean t-shirts, right? You use different spaces for different things.

Computers do the exact same thing with memory. Not all data is the same!

  • Your Age is a number (e.g., 15).
  • Your Name is text (e.g., "Ahmed").
  • Your Game Score is a number (e.g., 9500).
  • A Temperature is a decimal number (e.g., 36.5).
  • Whether your character is Alive or Dead is a true/false question.

Because these are all different, a programming language needs to know exactly what type of data you are trying to store. A Data Type is just a label that tells the computer: "Hey, this box is only for whole numbers!"

🏷️ Introduction to C++ Data Types

C++ has specific keywords for different data types. Here are the 5 most important ones you need to know:

int
Stands for Integer. Used for whole numbers.
Example: 10, -5, 1000
float
Used for decimal numbers.
Example: 3.14, 99.9, -0.5
char
Stands for Character. Stores a single letter or symbol inside single quotes.
Example: 'A', 'z', '?'
string
Used for text (multiple characters). Always inside double quotes.
Example: "TechTicket", "Hello!"
bool
Stands for Boolean. Can only be true or false.
Example: true, false

💻 Seeing Data Types in Action

Here is what all those different variable types look like when we type them into Visual Studio Code!

main.cpp 
#include <iostream>
#include <string>
using namespace std;

int main()
{
    // 1. Creating the variables with different types
    int score = 100;
    float temperature = 36.5;
    char grade = 'A';
    string name = "Ali";
    bool isWinner = true;

    // 2. Printing them out
    cout << score << endl;
    cout << temperature << endl;
    cout << grade << endl;
    cout << name << endl;
    cout << isWinner << endl;

    return 0;
}

Fun Fact: When you print a `bool` (true/false) in C++, the screen will usually show a `1` for true, and a `0` for false! Computers love binary.

🎤 User Input with cin

So far, our programs just talk to us using cout. But real programs are interactive—they listen! When you play a game, it asks for your username. When you use a calculator, it asks for numbers.

To make our C++ program listen to the keyboard, we use the cin (pronounced "See-In") command.

The Input Flow

💬 Your First Interactive Program

Let's write a program that asks the user for their name and age, and then talks back to them!

main.cpp 
#include <iostream>
#include <string>
using namespace std;

int main()
{
    // 1. Create empty boxes (variables) to hold the user's answers
    string name;
    int age;

    // 2. Ask the user for their name
    cout << "Enter your name: ";
    cin >> name;

    // 3. Ask the user for their age
    cout << "Enter your age: ";
    cin >> age;

    // 4. Print a custom message using their data!
    cout << "Hello " << name << endl;
    cout << "You are " << age << " years old!" << endl;

    return 0;
}

🔍 Code Breakdown

Let's look at exactly what happened above:

  • string name;
    We created a box named `name`. Notice we didn't put anything in it yet using =! It's completely empty, waiting for the user.
  • cout << "Enter your name: ";
    This prints a prompt on the screen so the user knows what they are supposed to do. If we didn't print this, the screen would just be a blank blinking cursor!
  • cin >> name;
    This is the magic line. The program pauses and waits for the user to type something and press ENTER. Whatever they type gets shoved into the `name` variable. Note that cin uses arrows pointing RIGHT >> (into the variable), while cout uses arrows pointing LEFT << (out to the screen).

🏧 Real-Life Analogy: The ATM Machine

Think of cin and cout like an ATM at a bank.

1. cout: The ATM screen says "Please enter your PIN."
2. cin: You type "1234" on the keypad.
3. The ATM stores "1234" in its memory to check if it's correct.

Your code is acting exactly like the brain of that ATM!

✅ Session Summary

  • Data Types tell the computer what kind of information is inside a variable.
  • int = whole numbers, float = decimals, char = single letter, string = text, bool = true/false.
  • cin is used to receive input from the user's keyboard.
  • The arrows for cin point RIGHT (>>), pushing data into the variable.
  • The arrows for cout point LEFT (<<), pushing data out to the screen.
