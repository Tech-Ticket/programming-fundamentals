Operators and Basic Calculations in C++

🎯 Session Overview

  • How to compare two pieces of data (like checking if a player's score is high enough to win).

⏪ Quick Recap

Let's remember the journey so far:

  • Variables (like int score;) hold data.
  • Data Types tell the computer what kind of data it is (numbers, text, etc.).
  • cin >> takes input from the keyboard and saves it in a variable.

We can store a user's age or a game's score, but what if the player finds a health potion? We need a way to add points to their health! This brings us to Operators.

⚙️ What Are Operators?

In programming, an operator is simply a special symbol that tells the computer to perform a specific mathematical or logical action.

You already use them every day without thinking about it! When you use a calculator, you press the + button. When you buy two items at a store, the cash register uses the + operator to find your total price.

If you write 5 + 3, the numbers 5 and 3 are your data, and the + is the operator doing the hard work to give you 8.

➕ Arithmetic Operators

C++ has five main symbols used for doing math. Most of them look exactly like the math you learn in school:

+

Addition

Adds two values together.
10 + 5 = 15

-

Subtraction

Subtracts the second value.
10 - 5 = 5

*

Multiplication

Notice we use an asterisk (*) not an x!
10 * 5 = 50

/

Division

Divides the first value by the second.
10 / 5 = 2

%

Modulus (Remainder)

Gives the remainder of a division.
10 % 3 = 1

🍕 Understanding Modulus (%)

Modulus sounds scary, but it's easy! Imagine you have 10 slices of pizza and 3 friends. If you divide them equally (10 / 3), everyone gets 3 slices. But there is 1 slice left over in the box. The modulus operator % simply tells you how many slices are left over! So, 10 % 3 = 1.

👀 Visualizing the Math Machine

Think of an operator as a factory machine. You feed it two ingredients (numbers), it processes them, and spits out a brand new product (the result).

💻 Simple Calculation Code

Let's write a program that creates two variables, adds them together, and saves the answer in a third variable!

main.cpp 
#include <iostream>
using namespace std;

int main()
{
    // 1. Create variables with our starting data
    int a = 10;
    int b = 5;

    // 2. Use the + operator and store the result in a new variable called 'sum'
    int sum = a + b;

    // 3. Print the result!
    cout << "Result: " << sum << endl;

    return 0;
}

🧮 Building an Interactive Calculator

Hardcoding numbers like 10 and 5 is okay, but real programs ask the user for numbers! Let's combine what we learned about cin in Session 3 with our new + operator to build a working calculator.

calculator.cpp 
#include <iostream>
using namespace std;

int main()
{
    // Create empty boxes for the user's numbers
    int num1;
    int num2;

    // Ask for the first number
    cout << "Enter first number: ";
    cin >> num1;

    // Ask for the second number
    cout << "Enter second number: ";
    cin >> num2;

    // Do the math!
    int result = num1 + num2;

    // Show the user the answer
    cout << "Sum = " << result << endl;

    return 0;
}

⚖️ Comparison Operators

Math isn't the only thing operators do. Sometimes, a program needs to compare two things. For example, verifying a password, or checking if a player has beaten the high score.

Comparison operators look at two pieces of data and produce a bool result: true or false.

  • > Greater than (Is 10 > 5? True!)
  • < Less than (Is 10 < 5? False!)
  • == Equal to (Notice it's TWO equal signs! One = is for putting data in a box, two == is for comparing them).
  • != Not equal to (The exclamation mark means "NOT").
  • >= Greater than or equal to.
  • <= Less than or equal to.

🎮 Real-Life Analogy: Game Logic

Video games use comparison operators constantly. Imagine fighting a boss.

The game constantly checks: bossHealth <= 0.
As long as that is false, the fight continues.
The moment it becomes true, the game plays the victory music and gives you loot!

🔥 Challenge Exercise: Calculate the Average

Write a program that calculates the average of two numbers entered by the user.

Math Reminder: To find the average of two numbers, you add them together first, and then divide the total by 2.

Example:
Enter first number: 10
Enter second number: 20
Average = 15
In C++, just like in regular math, you can use parentheses ( ) to force the computer to do the addition before the division!

int average = (num1 + num2) / 2;

✅ Session Summary

  • Operators are symbols that perform actions on variables and values.
  • Arithmetic operators (+, -, *, /, %) do mathematical calculations.
  • You can combine cin and operators to build interactive calculators.
  • Comparison operators (>, <, ==, !=, etc.) compare two values and result in true or false.
  • A single = is for assigning a value. Double == is for checking if two things are identical!
