🤔 The Problem: Programs Need to Remember

Imagine you are playing your favorite video game. You defeat a boss, collect 50 gold coins, and level up. How does the game know you have 50 coins instead of 0?

Or think about logging into Instagram. How does the app remember your username when you open it?

Programs are useless if they suffer from amnesia! They need a way to store data (information) so they can use it later. To do this, computers use something called Variables.

📦 What is a Variable?

In simple terms, a variable is a container that stores information.

Think of your computer's memory like a massive warehouse full of empty cardboard boxes. When you create a variable, you grab an empty box, put a label on the outside (the variable's name), and put some data inside (the value).

Visualizing computer memory: The label is the variable name, the item inside is the value.

💻 Creating Variables in C++

In C++, you can't just throw things into a box randomly. C++ is very strict! You have to tell the computer exactly what type of data is going to be stored in the box before you use it.

Rule to remember: Always end your variable creation with a semicolon ;!

🧩 Common Variable Types

Since C++ needs to know what "shape" the box is, we use different keywords for different types of data. Here are the big four you will use all the time:

int

Whole numbers (no decimals). Used for scores, health points, or age.

int score = 100;

float

Decimal numbers. Used for things like money, temperature, or precise math.

float temp = 36.5;

string

Text. Words, sentences, or characters. Always wrapped in double quotes " ".

string name = "Ali";

bool

True or False. Used for simple yes/no switches in your program.

bool isWinner = true;

🏠 Real-Life Analogy: The Room Organizer

Think about organizing your bedroom. You don't just throw everything in one giant pile. You use specific boxes:

  • A box labeled "Games" (String) that holds Minecraft and FIFA.
  • A piggy bank labeled "Money" (Float) that holds 15.50.
  • A light switch labeled "Light On" (Bool) that is either True (up) or False (down).

In C++, you are the organizer, and variables are your boxes!

🖨️ Printing Variables to the Screen

Storing data is great, but we want to actually see it! We can use our old friend cout to print variables just like we print text.

Important: When you print a variable, do NOT put quotes around the variable name. If you put quotes around "age", it prints the word "age". If you just type age, it prints the number inside the box!

main.cpp 
#include <iostream>
#include <string>
using namespace std;

int main()
{
    // 1. Create the variables
    string name = "Ali";
    int age = 15;

    // 2. Print them to the screen
    cout << name << endl;
    cout << age << endl;

    return 0;
}

Output:

Ali
15

🔍 Code Breakdown

Let's look closely at the new lines we just added:

  • #include <string>
    Because we want to use the string type to store text, we have to include this extra tool at the top of our file.
  • string name = "Ali";
    We grab a box meant for text (string), label it name, and put the word "Ali" inside it.
  • cout << name << endl;
    We tell the computer: "Look inside the box labeled name and print whatever is inside it." Then endl tells it to hit the Enter key to move to the next line.

✅ Session Summary

  • A variable is a labeled box in the computer's memory used to store data.
  • You must declare the type of a variable before using it in C++.
  • int is for whole numbers, float is for decimals.
  • string is for text (always use " "), and bool is for true/false.
  • You can print the contents of a variable using cout << variableName; (No quotes around the variable name!).
